# Internal utility functions Create record IDs for a batch of records (internal)

Internal utility functions Create record IDs for a batch of records
(internal)

## Usage

``` r
add_record_ids(interactions, author_lastname, publication_year)
```

## Arguments

- interactions:

  Dataframe of records

- author_lastname:

  Author lastname for ID generation

- publication_year:

  Publication year for ID generation

## Value

Dataframe with record_id column added
